## Installation

To get started with this project, clone the repository and install the dependencies:

```sh
git clone https://github.com/Tanush-J/NerdsBasement.git
cd NerdsBasement
npm install
```

## Scripts

To run the backend cd into the backend and run

```sh
npm start
```
